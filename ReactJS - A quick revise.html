Step: 1  React Components:
In react there are two types of components:
1. Class components.
2. Function components. (with hooks)

If you get a good grip about react components, designing web app in react is all about how you play with these components : )
Step: 2 General template of Class component and Function component:
1. Class component:
-------------------------------

constructor(props) {
    super(props);
    this.state = {"name":"manju", "age":"22"}    //creating state in welcome component
    props_one = this.props.attribute_one        //receiving props (here we are receiving two props: attribute_one 
    props_two = this.props.attribute_two          and attribute_two)

 }

***In recent updated react constructor became optional, so props and state can be accessed directly using:
this.state.state_variable  and this.props.attribute_name***

componentWillMount() {...action...}    //is invoked before the component is mounted (Like if you want to fetch the data using APIs from DB before rendering 
                                       or loading the page).

componentDidMount() {...action...}     //is invoked immediately after a component is mounted (means when page rendered(loaded) for the first time).

componentWillUpdate(nextprops, nextstate) {...action...}   //is invoked immediately after state updating occurs but before rendering.(Note: Never use this function to update state, 
                                         because it will results in recursive rendering(infinite loop))

componentDidUpdate(prevprops, prevstate) {...action...}   //is invoked immediately after state updating occurs but after rendering.(Note: Never use this function to update state, 
                                         because it will results in recursive rendering(infinite loop))

componentWillUnmount() {...action...}  //is invoked immediately before a component is unmounted and destroyed.

shouldComponentUpdate(nextProps, nextState) {...action...}  //is invoked before rendering when new props or state are being received.
                                                              Defaults to true. This method is not called for the initial render or when forceUpdate() is used.

***Still there are many life cycle functions are available (Most of them are Unsafe) in react but the above mentioned are the most commonly used life cycle functions***

2. Function component:
------------------------

import React, { useState } from 'react';

function Example() {
  // Declare a new state variables, which we'll call "count" and "name" 
  const [count, setCount] = useState(0);
  const [name, setName] = useState("manju"); // for each state variable we need to call useState() function; In this "name"
                                                is a state variable and "setName" is function that can be used to update the
                                                value of "name". In useState() initial value can be passed to set the initial
                                                value to state variable, ex: [variable, updateVariable] = useState("initialValue")
  
// Similar to componentDidMount and componentDidUpdate:  
  
useEffect(() => {    
   // Update the document title using the browser API    
   document.title = `You clicked ${count} times`;  
  });


// Only re-subscribe(execute) if props.friend.id changes (observe the array which is sent as second parameter to useEffect(),
in this case this useEffect() will be triggered only when the particular state or prop value changes which is sent in [ ])

useEffect(() => {
  function handleStatusChange(status) {
    setIsOnline(status.isOnline);
  }

  ChatAPI.subscribeToFriendStatus(props.friend.id, handleStatusChange);
  return () => {
    ChatAPI.unsubscribeFromFriendStatus(props.friend.id, handleStatusChange);
   };
}, [props.friend.id]);


***can use multiple useEffect() functions according to need in a single function component***

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}


Step: 3 useEffect() hook can be used as other react life cycle functions(didmount, didupdate, willunmount, etc), just it needs a simple syntactical changes in implementation.

useEffect(() => {
    console.log('componentDidMount');

    return () => {
        console.log('componentWillUnmount');
    };
}, []);

--------------------------------------------------------------

useEffect(() => {
        console.log('Component did update');
    });

    }, [count]);
